# Streamer face avatars

Drop a **square** image here named `<slug>.png` and it appears automatically in
the streamer selector (grid + recents + favourites). Until a file exists, that
streamer shows coloured initials as a fallback — no errors, no code changes.

- **Format:** PNG, square (e.g. 256×256). Non-square works but gets centre-cropped.
- **Naming:** must match the slug exactly (lowercase, no spaces). List below.
- After adding images, send them my way and I'll normalise them to a uniform
  square size/quality with a script if you want them consistent.
